Um codigo simples em C;
O programa recebe o input de dois numeros inteiros diretamente do usuario, Após 
isso o programa retorna a ele: A soma, A subtração, A multiplicação, A divisão e o resto entre os dois números;

o programa usa funções basicas da bibilhoteca [stdio.h];;
o atribuimento do valor da variavel pelo usuario é feito pela função scanf();

o programa também verifica qual dos dois numeros é maior e qual é o menor, assim, realizando a subtração de forma coerente.
